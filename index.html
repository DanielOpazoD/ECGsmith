Cómo leer este libro

Cada capítulo está pensado para 12–18 minutos. Las cajas de color son señales para detenerse: tip (verde) refuerza una regla práctica; perla (violeta) destaca un concepto sutil; alerta (ámbar) marca una trampa frecuente; caso (rojo) presenta una viñeta clínica con su lección.

Usa ← y → para navegar entre capítulos; el progreso de lectura se muestra en la barra superior. Los apéndices funcionan también como referencia rápida en consulta.
